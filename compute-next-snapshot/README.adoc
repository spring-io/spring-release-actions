= Compute Next Snapshot Action

This GitHub action computes the next snapshot version for a project.

== Inputs

|===
| Input | Description | Required

| `version`
| The current version.
| `true`
|===

== Outputs

|===
| Output | Description

| `version`
| The next snapshot version for the project.
|===

== Example Usage

[source,yaml]
----
- name: Compute Next Snapshot
  id: next-snapshot
  uses: spring-io/spring-release-actions/compute-next-snapshot@a1f321783a0769dd2aea4fad6c2ae2f95a52b885 # 0.0.5
  with:
    version: '1.2.3'

# Use the output from the `next-snapshot` step
- name: Echo next snapshot version
  run: echo "The next snapshot version is ${{ steps.next-snapshot.outputs.version }}"
----
